## 原文 配列 드릴 작동 방식  ★ v14.2 신규

족보에 인용된 한문 원문을 의미 단위 토막으로 나눈 뒤, 무작위로 섞어놓고 사용자가 정답 순서로 재배열:

**예시 (사용자 요청 원문 — 대시호탕 醫宗金鑑 평)**:
```
정답 순서:
[斯方也,] [柴胡] [得] [生薑之倍,] [解半表之] [功捷;]
[枳·芍] [得] [大黃之少,] [攻半裏之] [效徐,]
[雖云下之,] [亦下中之] [和劑也.]
```

### 인터랙션
| 동작 | 결과 |
|---|---|
| 풀에서 토막 클릭 | 첫 번째 빈 슬롯에 배치 |
| 슬롯의 토막 클릭 (채점 전) | 풀로 되돌리기 |

| 「해석 보기」 | 한글 의미 + 출전 표시 |
| 「초기화」 | 슬롯 비우기, 토막 다시 셔플 |
| 「다시 풀기」 (채점 후) | 같은 문제 재도전 |
| 「← 이전 / 다음 →」 | 처방간 이동 |

## 디자인 시스템

기존 의서궁 디자인 변수(`--zhusha` 朱砂, `--huang` 黃금, `--mi` 米색, `--feicui` 翡翠)와 폰트(`ZCOOL XiaoWei`, `Noto Serif KR/SC`)를 **그대로 재사용**.

- 헬게이트는 **朱砂 빨강** 배너 (압박감)
- 방감은 **黃금/古銅** 배너 (학구적 정리)
- 관계도는 **翡翠색** 배너 (시각적·구조적)
- 본초 칩은 13가지 기능별 색상으로 한눈에 구분 (解表 翡翠 / 淸熱 靑 / 補氣 黃 / 補血 紫紅 등)
- 君臣佐使는 별도 색상 라벨로 처방 카드와 매트릭스에서 일관 표시

## 호환성 / 안전성

- v13 헬게이트와 **완전 독립** — 헬게이트 → 방감 → 관계도 순서로 나란히 배너
- `FORMULAS`가 빈 배열이어도 `FALLBACK_COMPOSITION`으로 모든 처방 카드/매트릭스 렌더
- v13 한자→한글 토글과 호환 (`MutationObserver`가 새 컨텐츠 자동 변환)
- 모바일 가로 스크롤 매트릭스 지원 (`-webkit-overflow-scrolling:touch`)
- 관계도는 **touch-action:none**으로 핀치/팬을 직접 제어 (브라우저 기본 줌과 충돌 없음)
- 캐시 키 bump → SW 강제 재install → PRECACHE 재다운로드
- JSDOM 통합 테스트 통과 (탭 전환·키보드 단축키·SVG viewBox 검증 완료)
