# 한의학 마스터 v14 — 方鑑(방감) 패치

이 zip은 v13(헬게이트) 이후 `hanimaster-main` 프로젝트에 **덮어쓰기** 방식으로 적용하는 v14 패치입니다.

## 적용 방법

원본 `hanimaster-main/` 폴더(v13 적용된 상태)에 이 zip의 모든 파일을 **그대로 덮어쓰기** 하세요.

- 추가된 파일 (신규):
  - `data-v14-relations.js`        — 派生 系譜·용량 비교·五積 분해·함정 카드 등 매핑 메타데이터
  - `bangje-v14-pyori-bo-map.js`   — 方鑑 엔진 (UI 렌더 · 의서궁/홈 배너 주입)

- 패치된 파일 (덮어쓰기):
  - `index.html`                   — v14 스크립트 2줄 추가 (v13 직후)
  - `sw.js`                        — 캐시 키 bump (v14-fangjian) + v14 파일 precache + 상단 주석 변경 이력

- 변경 없음 (확인용):
  - `data-formulas.js`             — 기존 FORMULAS 11처방 그대로 활용 (방감은 이 데이터 위에 메타만 얹음)
  - `bangje-v13-hellgate.js`       — v13 헬게이트 그대로
  - `bangje-v13-integration.js`    — v13 통합 패치 그대로 (배너 패턴 차용만 함)

1. **의서궁(醫書宮)** 첫 화면 — 시험 D-N 패널 아래, 헬게이트 배너 **바로 다음**에 黃금색 「方鑑 NEW v14」 배너

## 方鑑 화면 구성

| 섹션 | 내용 |
|---|---|
| **分** 시험범위 五分類 | 해표공리·해표청리·해표온리·보기·보혈 표 (병증×치법×방제) |
| **方** 處方 카드 | 7장 6방 + 8장 7방 = **13 主要方** 카드. 각 카드에 君臣佐使 색상 코딩된 본초 칩 + 작용·주치·키포인트 (FORMULAS의 keyPoints 활용) |
| **系** 派生 系譜 | 大柴胡 공식, 小承氣 vs 厚朴三物 용량차, 石膏湯, 桂枝人蔘湯, 四君子 一家 7파생, 보중익기 4파생, 사물탕 5파생 — **8개 系譜** |
| **量** 용량 比較 | 大柴胡·小柴胡·小承氣·조위승기·후박칠물·후박삼물 비교표 / 당귀보혈탕 5:1 비율 |
| **矩** 處方 × 本草 매트릭스 | 가로 본초(기능별 색상) × 세로 13처방, 君臣佐使 색상 칠 |
| **五** 五積 분해 | 寒·濕·痰·氣·血積 색상 카드 + 22객 함정 "기적-궁귀탕" 경고 |
| **較** 비교 카드 | 大柴胡 vs 大靑龍 vs 石膏湯 / 백호탕 vs 당귀보혈 / 사군자·삼령·보중익기 / 사군자 4파생 |
| **罠** ★ 함정 카드 | 22~18 기출의 자주 틀리는 11문항 (객관식 함정 답안 포함) |

## 디자인 시스템

기존 의서궁 디자인 변수(`--zhusha` 朱砂, `--huang` 黃금, `--mi` 米색)와 폰트(`ZCOOL XiaoWei`, `Noto Serif KR/SC`)를 **그대로 재사용**.

- 헬게이트는 **朱砂 빨강** 배너 (압박감)
- 방감은 **黃금/古銅** 배너 (학구적 정리)
- 본초 칩은 13가지 기능별 색상으로 한눈에 구분 (解表 翡翠 / 淸熱 靑 / 補氣 黃 / 補血 紫紅 등)
- 君臣佐使는 별도 색상 라벨로 처방 카드와 매트릭스에서 일관 표시

## 호환성 / 안전성

- v13 헬게이트와 **완전 독립** — 헬게이트 배너 → 방감 배너 순서로 나란히 표시
- `FORMULAS`가 빈 배열이어도 `FALLBACK_COMPOSITION`으로 모든 처방 카드/매트릭스 렌더
- v13 한자→한글 토글과 호환 (`MutationObserver`가 새 컨텐츠 자동 변환)
- 모바일 가로 스크롤 매트릭스 지원 (`-webkit-overflow-scrolling:touch`)
- 캐시 키 bump → SW 강제 재install → PRECACHE 재다운로드
