# CHANGELOG v14 — 方鑑(방감) 패치

**릴리즈:** 2026-05-19
**기반:** v13 헬게이트(獄門) 위에 누적 적용

## 한 줄 요약

2차 수시 시험범위(7장 表裏雙解劑 + 8장 補益劑)의 13 主要方을 한 화면에서 시각적으로 매핑하는 **「方鑑」** 모드 추가.

## 추가된 컨텐츠

### 1. 처방-본초 매트릭스
- 가로축 본초(기능별 색상 코딩 13종) × 세로축 13 처방
- 君臣佐使 색상으로 칠해진 셀 (君=朱砂, 臣=古銅, 佐=翡翠, 使=玄)
- 모바일 가로 스크롤 지원, sticky header/row

### 2. 派生 系譜 8종
- 大柴胡湯 구성 공식 (소시호 − 인삼·감초 + 소승기 − 후박 + 작약)
- 小承氣湯 vs 厚朴三物湯 (같은 3味, 다른 용량 → 다른 主治)
- 石膏湯 구성 (황련해독탕 + 석고 + 마황·향시)
- 桂枝人蔘湯 (이중탕 + 계지)
- **四君子湯 一家 7파생** (이중→사군→이공→육군→향사육군→보원·삼령·보중익기)
- 補中益氣湯 4파생 (거원전·승함탕·익기총명탕·소음인 보중익기)
- **四物湯 5파생** (교애탕→사물→도홍·금련·계부·팔진)

### 3. 五積散 五積 분해 카드
- 寒·濕·痰·氣·血積을 색상 카드로 분리
- 각 적의 본초 + 기본방 합방 명시
- ★ 22객 함정 "氣積-궁귀탕" 오답 경고 표시

### 4. 비교 카드 4종
- 大柴胡湯 vs 大靑龍湯 vs 石膏湯 (모두 表+裏 동시)
- 白虎湯 vs 當歸補血湯 (경고: 잘못 백호 쓰면 必死)
- 四君子湯 · 參苓白朮散 · 補中益氣湯
- 사군자 4파생 (이공·육군·향사육군)

### 5. ★ 함정 카드 11종 (22~18 기출)
- 22학번 객관식 함정 답안과 힌트를 모두 명시
- 각 카드에 학번/처방/문제/정답/힌트 4단 표시
