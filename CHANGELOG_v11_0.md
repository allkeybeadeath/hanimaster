## 5. 진입 방식

### 신규 사용자 (S.seenHub 미설정)
- 첫 PWA 로드 시 자동으로 醫書宮 hub 표시 (600ms 지연 후)
- `S.seenHub = true` 저장 → 다음부터는 자동 진입 안 함

### 기존 사용자 (lastTab='home' 등)
- 종전 그대로 방제학 home 으로 진입 (UX 무변경)
- 헤더 좌측 「宮」 chip 클릭 → hub 진입

### 어디서나 「宮」 chip
- header의 head-chips 영역 맨 앞에 자동 inject
- 모든 화면에서 항상 표시 → 1탭으로 의서궁 복귀

## 7. 알려진 한계

- **방제학 외 모든 방은 콘텐츠 미주입** — UI shell 만. 강의 자료 제공 후 본격 구축.
- **하단 네비게이션은 방제학 전용 그대로** — 6 tabs (대청·처방·약재·기출·통계·명예) 가 방제학 컨텍스트로만 작동. 다른 방 안에서 nav 탭 클릭 시 방제학으로 이동 (의도된 動作).
- **氣·등급·캐릭터·인장·업적·인체 시각화** 는 모든 방에서 공유 — 한 계정 = 한 정체성.
- **린도인 사진** 은 唐僖宗 인물도를 차용 — 실재 초상 부재. 향후 더 적절한 이미지 발견 시 교체 권장.
- **사암도인 사진** 은 사용자 제공 수묵 後背圖 — 익명/추상 표현. 실제 초상이 존재하지 않으므로 의도된 선택.

## 8. v11.1+ 향후 계획

1. 진단학 콘텐츠 주입 — 강의 자료 받아 사상체질 진단법·四診·辨證 데이터·문항 추가
