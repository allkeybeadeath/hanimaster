# 한의학 마스터 v13 — 헬게이트(獄門) 패치

이 zip은 기존 `hanimaster-main` 프로젝트에 **덮어쓰기** 방식으로 적용하는 v13 패치입니다.

## 적용 방법

원본 `hanimaster-main/` 폴더에 이 zip의 모든 파일을 **그대로 덮어쓰기** 하세요.

- 추가된 파일 (신규):
  - `data-hellgate.js`         — 헬게이트 데이터 (24 처방, 124 문제)
  - `bangje-v13-hellgate.js`   — 헬게이트 엔진 (UI · 채점 · 동적 상태)
  - `bangje-v13-integration.js`— 의서궁·홈·기출 통합 + 한글 토글 + CIM Lab 런타임 제거

- 패치된 파일 (덮어쓰기):
  - `index.html`               — v13 스크립트 로드 추가
  - `sw.js`                    — 캐시 키 bump + v13 파일 precache
  - `data-questions-bulk.js`   — 91개 leaky 문제의 짧은 distractor 패딩 + CIM Lab 제거
  - `data-formulas.js`         — 4개 leaky 문제 distractor 패딩
  - `bangje-v98-home.js`       — CIM Lab 제거
  - `bangje-v98-dictplus.js`   — CIM Lab 제거
  - `admin-reset.html`         — CIM Lab 제거

- 변경 없음 (확인용):
  - `bangje-v11-clinic-hub.js` — 원본 그대로 (런타임 후킹으로 헬게이트 배너 삽입)

## 헬게이트 기능 요약

| 항목 | 내용 |
|---|---|
| 문제 수 | 124개 (목표 100문제 초과) |
| 문제 유형 | 구성·작용·주치·증상·배오원리·가감·용량·비교·서술 |
| 답안 형식 | 100% 주관식·서술형 (객관식 없음) |
| 띄어쓰기 | 무관 (예: `황기감초인삼승마...` 인식 ✓) |

| 오타 | 1글자 오타까지 허용 (Levenshtein ≤ 1) |
| 약식 표기 | 계감(=계지·감초), 삼황(=황금·황련·황백), 금련, 교애, 강조, 궁귀 등 |
| 동적 채점 (구성) | 8미 ↑ 처방은 8개만 적기 → 맞춘 약재는 다음 회차 채점기준에서 제외 → 모두 맞히면 롤백 |
| 동적 채점 (증상) | 3개만 적기 → 같은 방식으로 갱신·롤백 |
| 반복 모드 | 논스톱 (큐 끝나면 셔플 후 재시작 — 2시간 전사 모드) |
| 종합 | 세션 정·오 카운트, 분 단위 경과 시간 |

## 추가 기능

- **기출(問) 탭 「전체 풀기」 버튼** — 문제수 선택 영역에 추가됨
- **<자동> 문제 필터** — 이름·출전·의가 묻는 자동 객관식 제거됨
- **난이도 3·4 distractor 강화** — 정답을 닮은 한 글자 swap 오답 자동 생성
- **한자→한글 토글** — 의서궁 우상단의 「韓字」 버튼 (전체 컨텐츠 한글 변환)
- **CIM Lab 제거** — 정적 파일 + DOM 런타임 모두 제거
