# v10.0.7 (2026-05)

룰 UI 갱신 — v10.0.4~v10.0.6 의 出牌 룰 변경 사항을 사용자가 게임 시작 전과 진행 중
모두에서 명확히 볼 수 있도록 표시.

## 변경 위치

### 1. 로비 「遊戲規則」 카드 (`bangje-cube.js` renderLobby)
기존 한 줄 "유효 set = 완성 방제 · 派生方 · 증상별 加減" 을 다음과 같이 확장:

- 첫 줄에 4 종류 모두 명시: "완성 方劑(基方) · 派生方 · 증상별 加減 · base 의 부분집합"
- 별도 강조 박스 (zhusha-d 좌측 보더, 옅은 배경) 로 「**出牌 룰 (v10.0.6)**」 섹션:
  - **初手** (붉은색) — 완성된 基方 강제, 사이즈 무관, 派生·加減·부분집합 불가
  - **續手↑** (옥색) — 完成 方劑 · 派生方 · 加減方 · base 의 부분집합 (≥3 본초, 어떤
    처방의 구성본초로만 구성) 모두 OK

### 2. 인게임 「조작법」 모달 (`bangje-cube.js` bindHelpAndLeave)
「終」 턴 종료 항목 아래에 동일한 構造의 룰 박스 인라인 삽입. 또 예시 한 줄 추가:
"續手 부분집합 예: 인삼·백출·복령 (사군자 3미) 출패 → 「사군자탕 부분」 으로 보드에
올림".
