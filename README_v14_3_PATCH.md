## 1) 原文 配列 드릴 (`bangje-v14-drill.js`)

### 실시간 채점
- **「채점 ✓」 버튼 삭제** — 토막을 슬롯에 놓는 순간 즉시 색상으로 정/오 판정
- 채워진 토막마다 초록(정답) / 빨강(오답) 즉각 표시
- 모든 슬롯이 정답으로 채워지면 자동으로 完成 상태 진입 → 통계 +1 / 다음 버튼 활성화

### 첫 구절 도입부 (始)
- 각 원문의 **0번 토막(첫 구절)은 슬롯에 미리 박힌 상태(locked)** 로 시작
- 우상단에 `始` 뱃지 + 황금 테두리 — "여기서부터 시작"이 시각적으로 명확
- locked 슬롯은 클릭해도 되돌릴 수 없음 (실수 방지)
- 풀(pool)에서도 첫 구절이 제외되어 단서로 작용

### 같은 글자 인터체인저블
**채점 기준이 "인덱스 매칭"에서 "텍스트 매칭"으로 변경:**
```
이전: STATE.slots[i] === i  (정확히 그 토막이 그 위치에 들어가야 정답)
지금: tokens[i] === tokens[STATE.slots[i]]  (그 위치의 텍스트와 일치하면 정답)
```
예: `'得'` 토막이 두 개 있으면 어느 자리든 `得` 자리에 들어가기만 하면 둘 다 정답.
의미 단위로 동일 어구가 반복되는 한문 문장에서 사용자가 헷갈리는 실수가 사라짐.

## 5) 처방 탭 확장 (`data-formulas.js`)

### 추가 3개

#### 玉屛風散 (옥병풍산) — 8장 補益劑·補氣
- 출전: 醫方類聚 (究原方 引)
- 구성: 黃耆·白朮·防風 (3味)
- 작용: 益氣固表止汗
- 적응: 表虛自汗·易感風邪 (黃耆 : 防風 = 2~3:1, 補多於散)
- 配伍妙用: "黃耆 得 防風 而固表不留邪, 防風 得 黃耆 而散風不傷正"

#### 生脈散 (생맥산) — 8장 補益劑·補氣
- 출전: 醫學啓源 — 張元素
- 구성: 人蔘·麥門冬·五味子 (3味)
- 작용: 益氣生津, 斂陰止汗
- 핵심: 一補(人蔘)·一淸(麥門冬)·一斂(五味子) — 暑熱傷氣陰의 대표방
- 적응: 暑熱이 氣陰을 傷한 證, 久咳肺虛 氣陰兩虛

#### 桃紅四物湯 (도홍사물탕) — 8장 補益劑·補血
- 출전: 醫宗金鑑 — 四物湯 加 桃仁·紅花
- 구성: 熟地黃·當歸·白芍·川芎·桃仁·紅花 (6味)
- 작용: 養血活血, 化瘀調經
- 적응: 血虛兼血瘀證, 婦人 經行腹痛·血塊紫黯
- 기출 핵심: 적응증에 「白帶證多」는 포함되지 않음

이로써 시험범위 8장 補益劑가 다음과 같이 완비:
