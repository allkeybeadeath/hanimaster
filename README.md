# 方劑學 学習帳 — 2차 수시 시험 대비 (5/20)

CIM Lab 본과 2학년 2학기 방제학 2차 수시 시험 대비용 PWA.

## 시험 정보

- **시험일**: 2026년 5월 20일 (화)
- **범위**:
  - 첫째 PDF의 1~59 페이지 = **제8장 보익제** (1절 보기제, 2절 보혈제, 3절 기혈쌍보제)
  - 둘째 PDF의 127 페이지~끝 = **제6장 온경산한제** 일부 + **제7장 표리쌍해제** 전체
- **총 24 처방** (23 핵심 + 향사육군자탕 등 가감 부속 일부 포함)
- **68 약재** 효능 요약
- **작년 기출 23 문** (22학번 1차수시 + 1학기 기말 중 범위 일치 항목만)

## 기능

- **홈·계획**: 시험까지 카운트다운, 4일 단계별 학습 계획 (5/17~5/20)
- **처방 24**: 처방별 작용·적응증·구성·군신좌사·기본방·핵심 포인트·가감법·기출 포인트
- **암기카드**: 3 모드 (작용·구성·적응증), 안다/모른다 추적
- **객관식**: 작년 기출 23 + 자동 생성(작용·적응증·구성 매칭) 합 80+ 문제
- **작년 기출**: 처방별 묶음 + 해설
- **처방 비교**: 헷갈리는 처방 쌍 비교표 (10 비교)
- **약재**: 68 약재 효능 + 사용 처방 역인덱스
- **오답함**: 자동 누적, 모아 풀기
- **함정 카드**: D-day 빈출 함정 19 항목

## 단계별 학습 계획

### D-3 (5/17 토) — 1단계 · 골격 파악
- 처방 24개 한자/한글 이름 외우기
- 8장 보익제·6장 온리제·7장 표리쌍해제 절 구조
- 처방 카드 1회 통독 (작용·적응증 한 줄)

### D-2 (5/18 일) — 2단계 · 구성·작용·적응증 암기
- 각 처방 구성약물 정확히 외우기 (특히 사군자·사물·팔진·십전·자감초)
- 작용(治法) 핵심 어구 (益氣健脾, 補血和血, 溫經散寒 등)
- 적응증 핵심증상 1~2개씩

### D-1 (5/19 월) — 3단계 · 기본방·가감·비교
- 합방 관계: 사군자+사물=팔진, 팔진+황기육계=십전, 계지탕 가족 5처방
- 처방 비교: 사군자↔향사육군자, 보중익기↔귀비, 자감초↔생맥산 등
- 작년 기출 24 문제 완답 + 해설

### D-day (5/20 화) — 4단계 · 최종 점검
- 함정 카드 19 항목 빠르게
- 빈출 비율·세부: 황기:당귀=5:1, 대시호탕 약량 등
- 오답함만 30분 훑기

## 배포 방법

단일 정적 디렉토리이므로 어디서나 배포 가능합니다.

### GitHub Pages
1. 저장소 루트에 모든 파일 배치
2. Settings → Pages → main 브랜치 source
3. 1~2분 후 `https://{user}.github.io/{repo}/` 에서 접속

### Netlify Drop
1. https://app.netlify.com/drop 접속
2. 폴더 드래그
3. 즉시 URL 발급

### 로컬 빠른 실행 (시험 직전, 인터넷 없이도)
```bash
cd paideia-bangje-v1
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000/
```

## 데이터 출처

- 시험 범위 PDF 두 개 (Hwp 2020 변환): 자동 추출 + 핵심 필드 직접 큐레이션
- 작년 기출 HWP 두 개 (22학번 1차수시 보익제~고삽제, 1학기 기말 4장5절~7장): 시험 범위와 겹치는 항목만 추출
- 한자(번체) 표기 우선. 한자 옆 한글 발음 첨자는 PDF 변환 잔재로 제거.

## 학술적 한계 (정직 명시)

- **자동 큐레이션 한계**: PDF의 한자+한글 첨자 레이아웃을 자동 처리하다가 일부 표(table) 구조에서 가독성이 떨어질 수 있습니다. 처방 상세의 [구성][작용][적응증][군신좌사] 핵심 필드는 수동 검증되었지만, [가감법] 일부 항목은 원문 그대로일 수 있습니다.
- **자동 생성 객관식**: 작용·적응증·구성 매칭 문제는 distractor 가 같은 처방 모음에서 무작위로 뽑힙니다. 실제 시험 출제 패턴과 다를 수 있으나, 학습 효과는 비슷합니다.
- **작년 기출 매핑**: 22학번 기출 중 올해 시험 범위(8장 보익제·6장 온경산한제·7장 표리쌍해제)와 겹치는 항목만 포함했습니다. 작년 시험 범위에는 더 많은 처방(육미지황·우귀환·고삽제·청장부열제 등)이 있었으나 올해 범위 밖이므로 의도적으로 제외했습니다.
