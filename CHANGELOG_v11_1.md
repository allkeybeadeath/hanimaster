## 3. 학습 모드 4종

### 객관식 (4지선다)
- 사진 + 「이 사진의 設體 특징은? / 舌色은? / 舌苔는? / 辨證은?」 중 하나 자동 출제
- 4 옵션: 정답 1 + 同類 풀에서 셔플된 distractor 3
- 답 선택 시 즉시 정답·해설 (한자 라벨 + 한글 + 변증 + 교재 페이지 + notes)
- 정답률·진행 상황 상시 표시

### 주관식
- 직접 입력 (한자 또는 한글 모두 허용)
- 채점: 정확 일치 + 한자↔한글 매핑 (氣虛↔기허 등) + 키워드 일치
- 시험 직전 한자 표기 연습용

### 드릴 (속습)
- 객관식과 동일하되 자동 진행 (정답 0.8초, 오답 1.8초)
- 빠른 반복 — 점수보다 노출 빈도

### 사진첩 (圖鑑)
- 48장 전체 펼쳐 그리드
- 「라벨 보기/숨기기」 토글 — 처음에는 가린 채 사진만으로 추측 → 라벨 공개 검증

각 모드는 **범위 탭** (설체/설질/통합) 으로 사전 필터링.

## 7. 알려진 한계

- **소요 시간 추정만** — 시험 시 실제 출제 방식 (사진 1장 보고 답 적기 vs 다중 사진 비교) 미반영. 객관/주관 모두 단일 사진 1문 기준.
- **사진은 교재 圖-1~圖-8 한정** — 더 많은 임상 사진 필요 시 추가 자료 주입 권장.
- **辨證 답안 풀** — 일부 변증명은 교재 라벨 그대로 사용 (예: '熱毒/血瘀'). 시험 답안 형식이 다르면 사용자가 응시 시 자체 변환 필요.
- **사진 라벨 정확도** — 교재 캡션 그대로 사용. 일부 사진 (㉘ 絳舌 등) 은 변증이 명시되지 않아 표준 진단학에 따른 추정 적용.
